# Compilación
Si deseas compilar el documento por ti mismo, puedes usar el script `compile.sh`, o compilar de cualquier otra forma que prefieras.

Para contribuir al proyecto **no se requiere que compiles el documento**, esto se hace automaticamente cuando se hace un push a la rama principal. Recomendamos incluir en tu archivo `.gitignore` la línea `*.pdf` para evitar subir archivos pdf al repositorio.
